# AEC Agent Skills

Specialized AI agent skills for Architecture, Engineering, and Construction (AEC) professionals. This project explores how AI can augment workflows in the built environment, providing tools to automate complex, domain-specific tasks and improve decision-making.

## Key Features
*   **Domain-Specific Task Automation:** Implements skills for common AEC workflows like document parsing and analysis.
*   **Modular Skill Design:** Allows for easy integration and extension of new capabilities.
*   **Agent-Oriented Architecture:** Built to function as part of a larger AI agent system.

## Tech Stack
*   Python
*   LangChain / LangGraph
*   OpenAI API

## Getting Started
1.  Clone the repository: `git clone https://github.com/zoreanuj/AEC_Agent_Skills.git`
2.  Install dependencies: `pip install -r requirements.txt`
3.  Set your OpenAI API key: `export OPENAI_API_KEY='your-key'`
